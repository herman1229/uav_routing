# A3C多GBS无人机路由优化系统

## 项目简介

本项目实现了基于A3C算法的多无人机协同路由优化。**核心特点：在一个Episode内依次完成3个GBS的路由决策**，模拟真实的联邦学习参数聚合场景。

## 快速开始

### 测试环境
```bash
python A3C_UAV_Routing_Env.py
```

### 开始训练
```bash
python A3C_UAV_Routing.py
```

## 核心特性

### 1. Episode = 3个GBS完整决策
```
1个Episode = GBS0决策 + GBS1决策 + GBS2决策
总奖励 = 3个GBS奖励之和
```

### 2. 负载依次累积
```
GBS0传输 → 更新网络负载
GBS1传输 → 看到GBS0的影响 → 更新网络负载  
GBS2传输 → 看到GBS0和GBS1的影响 → 更新网络负载
```

### 3. 成功/失败区分
```
成功(到达Server): 更新网络负载 + 获得正奖励
失败(超时100步): 不更新网络 + 惩罚-10
```

### 4. 有向网络拓扑
```
GBS → Router: 单向（数据上传）
Router ↔ Router: 双向（数据转发）
Router → Server: 单向（数据汇聚）
```

## 训练参数

- 总Episodes: 1200（每个包含3个GBS）
- Worker数: 8
- 每个GBS最大步数: 100
- 节点容量: 50流表项
- 学习率: 1e-3

## 预期效果

### 奖励范围
- 最佳: 24（3个GBS都成功，2跳）
- 良好: 15-20（3个GBS都成功，3-4跳）
- 一般: 0-10（2个成功，1个失败）
- 较差: -30到0（多个失败）

### 成功率
- 初期: 40-60%
- 中期: 70-85%
- 后期: 90-100%
